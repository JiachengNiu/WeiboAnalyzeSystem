Abstract

Niu Jiacheng (Information Management and Information System)

With the rapid development of the Internet, Sina Microblog as a new generation of network media, has become a distribution center of mass information. On this basis, Sina Microblog set up a real-time hot search list based on users’ searching data. The list reflects the real-time hotspot objectively and becomes an important channel for users to get hotspot information. By analyzing the real-time hotspot of Microblog, we can provide users with higher-quality hotspot information service, and reach the goal of hotspot pursuit and public opinion monitoring.

This paper is exactly the design and implementation of a Microblog real-time hotspot analysis system. This paper investigates Microblog analysis systems, real-time hotspot analysis methods and short text classification algorithms at home and abroad, then designs the classification algorithm suitable for Microblog real-time hotspots, realizes the automatic classification of Microblog hotspots. After that, this paper designs the functions, the framework and the database structure of this system, divides the system into five modules: Data acquisition module, data storage module, machine learning module, data analysis module and front-end display module. Finally, the whole system is realized with bootstrap front-end frame and web.py back-end framework.
